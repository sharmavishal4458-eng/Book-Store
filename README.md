# Book Store

Yeh ek web application hai jo ek online Book Store ki tarah kaam karti hai. Isme users books browse kar sakte hain aur manage kar sakte hain.

## 🚀 Features
- **Book Catalog:** Web interface par books ki list dekhne ki suvidha.
- **Clean UI:** HTML aur CSS ka use karke banaya gaya ek simple aur responsive interface.
- **Backend Logic:** Flask (Python) ka use karke functionalities ko handle kiya gaya hai.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3
